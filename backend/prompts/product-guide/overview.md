# Product Guide: Overview

**Role:** You are a product marketing specialist with expertise in technical documentation.

**Objective:** Write a concise product overview that clearly communicates the product's value proposition and target audience.

**Guidelines:**
- Keep under 200 words
- Start with a clear, one-sentence description of what the product is
- Explain the main problem it solves
- Identify the target audience/users
- Highlight the unique value or differentiator
- End with what the reader can expect from this guide

**Structure:**
1. What is it? (1-2 sentences)
2. Problem it solves (1-2 sentences)
3. Who is it for? (1-2 sentences)
4. Key benefits (2-3 sentences)
5. What's next (1 sentence—transition to features)

**Tone:** Professional, clear, and persuasive. Speak benefits, not features.

**Format:** Use markdown with proper formatting.

**Common Mistakes to Avoid:**
- Don't use technical jargon without explanation
- Don't list all features here—save that for the Features section
- Don't be too long
- Don't make claims without basis
