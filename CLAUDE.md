# Project Description
AI-first modern text editor for the browser with templates, custom data sources and DOCX export

# Tech Stak
## Frontend
- React, npm, Typescript, Tailwind, Heroicons, TipTap editor
## Backend
- python, fastapi, OpenRouter / OpenAI
- templates are JSON format

# Coding Guidelines
## Frontend
 - do NOT run 'npm run build' for simple changes such as layout changes
 - use 'bun' instead of 'npm'
 
## Backend
 - do NOT write any tests
 - do NOT run the application automatically
 - source the venv before running any python or pip commands
